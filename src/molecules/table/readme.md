# Tables

The basic scrollable table starts scrolling below the <strong>medium</strong> breakpoint (<strong>769px</strong>).

Responsive variants to compare:

- <strong>Scrollable</strong>: keeps the full table and adds directional overflow hints only when horizontal scrolling is actually needed.
- <strong>Stacked</strong>: turns each row into a card with column labels above each value.
- <strong>Sticky first column</strong>: pins the first column while the rest of the table scrolls horizontally.
